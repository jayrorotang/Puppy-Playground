# 🐶 Puppy Playground

一个用**鼠标操控**的互动小狗网站 —— 你不是在「看」视频,而是在「指挥」它。

🔗 **在线体验:** https://jayrorotang.github.io/Puppy-Playground/

> 桌面浏览器 + 鼠标体验最佳。移动端会以缓慢循环自动播放作为兜底。

---

## ✨ 玩法

全屏铺满一只小狗的视频,**没有播放键、没有进度条**——移动鼠标,故事就发生。

| 场景 | 交互 |
|---|---|
| 🍖 **Feeding** | 左右移动鼠标,小狗跟着你转头(空间 → 时间) |
| 🖐️ **High-five** | 上下移动:上=正放,下=倒放 |
| ⚽ **Running** | 左右移动追着球跑(0.8 倍慢放,更好控制) |
| 😴 **Sleeping** | 黑屏 + 呼吸声,拉一下灯绳点亮房间,苍蝇指针把小狗「挠」醒 |

- **光标即角色**:每个场景的指针是小狗会回应的道具——饼干、球、手掌、苍蝇 🪰。
- **氛围声景**:四个页面各有一段程序化合成的循环背景音乐;睡觉页黑屏有柔和呼吸声,开灯后是苍蝇嗡嗡声。
- **拉绳灯开关**:点击灯泡,绳子会真实地伸缩回弹,灯光渐亮、房间淡入,再点一下回到黑屏。
- **蜡烛光晕**:黑屏时鼠标周围有跟随的暖色光圈,带轻微摇曳。
- **按钮音效**:菜单点击、悬停、滚动、开关都有克制的合成音效。

## 🛠 技术亮点

- **纯前端单文件**:整站就是一个 `index.html`(HTML + CSS + 原生 JS),无框架、无构建。
- **视频 scrub**:用 `video.currentTime` 做逐帧 seek,配合 seek 锁避免频繁卡顿;鼠标位移映射为时间轴增量,支持正反双向。
- **Blob 加载**:视频以 `fetch → createObjectURL` 整段载入,绕开服务器 Range 限制,seek 完全本地进行。
- **Web Audio 合成**:所有背景音乐、环境音和按钮音效都是程序化合成,**无外部音频文件、无版权问题**。
- **响应式**:大量 `clamp()` 流式尺寸 + 宽度感知的弧形菜单布局 + `100dvh`,从手机到 4K 平滑适配。

## 📁 结构

```
index.html   # 整个应用（样式 + 弧形菜单 + 视频/音频逻辑）
dog.mp4      # Feeding（转头）
dog3.mp4     # Running（奔跑）
dog4.mp4     # High-five
dog5.mp4     # Sleeping（睡觉）
*.png        # 光标图（饼干 / 球 / 手掌）
```

## 🚀 本地运行

因为视频用 blob 加载,需要通过本地服务器打开(不能直接双击 `index.html`):

```bash
python3 -m http.server 8123
# 然后访问 http://localhost:8123
```
